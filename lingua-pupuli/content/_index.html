The Lingua Pupuli Project

Language of the Puppet

lingua (nominative) : Language
pūpulī (genitive singular) : Puppet
When we first started to create the Puppet VS Code extension everything could be contained in a single git repository. However as time went on, it became apparent that we had out-grown a single repository, and so, we created the Lingua Pupuli project.

This project is an umbrella for a community of tools; Tools to help make using, and writing, Puppet faster and easier. It is composed of three projects; VS Code Extension, Editor Services and Editor Syntax.
